### Install Tutorial

Before you begin, ensure you have met the following requirements:

* [Git](https://git-scm.com/downloads "Download Git") must be installed on your operating system.

### Run Locally

To run **Password Manager** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/zaxxdev/password-manager.git
```

Windows:

```bash
git clone https://github.com/zaxxdev/password-manager.git
```
